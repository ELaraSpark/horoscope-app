## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/horoscope-app.git
cd horoscope-app

2. Install Dependencies
Frontend:
cd hooroscope-app/
npm i

Backend :
cd ../backend
npm install

3. Run the App (Dev Mode)
Backend:
cd backend
npm run dev

Frontend:
cd frontend
npm run dev
